GENERAL GUIDELINES
  • The game MUST be able to work within an html context using the following keyboard keys
    1. W,A,S,D
    2. ARROW KEYS
    3. K
    4. L
    5. Enter
    6. Shift
  • It must have an aspect ratio of 1:1 (Square)
  • It must have at least one user action (like clicking a menu button and etc) before triggering any sound (browser rules)
GAME RESOLUTION

The game MUST be a 300x300 to fit properly

GAME IMAGE
  • The game image MUST be named with the same name as the game
  • The image MUST be in the root folder where your index.html file is
  • It must be a PNG file
GAME CONTROLS

Your game must follow the classic game boy control pattern

  • UP, LEFT, DOWN, RIGHT for movement
  • A for positive actions
  • B for negative actions
  • START for START actions
  • SELECT for SELECT actions
